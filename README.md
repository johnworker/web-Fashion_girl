# 說明

## 網站說明

排版 | 說明
-------------- | --------------
 nav | (由左至右依序為) 首頁Logo、連結分頁、社群連結
 Carousel | 主頁使用輪播效果 
 RECOMMEND | 推薦產品(藉此把冷門商品重新做組合販售)
 New Products | 推薦新產品(初步把新品放最後，以後可調動位置)
 cart | 每個產品圖片點選後接可連至購物車分頁
 add to cart | 點選後加入購物車，再點選購物車icon決定採購數量
